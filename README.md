# 🐾 Find MyPet (BuscaPet)

Um ecossistema de rastreamento de pets de baixo custo e alta eficiência energética. 

Diferente de soluções de mercado que dependem de redes colaborativas (mesh) de terceiros, o Find MyPet utiliza uma arquitetura IoT autônoma. O dispositivo vestível (coleira) identifica zonas seguras através de redes Wi-Fi domésticas para poupar bateria e aciona módulos de GPS e conectividade móvel (GSM/4G) apenas em situações de fuga, enviando as coordenadas em tempo real para o aplicativo do tutor.

### 🚀 Tecnologias Utilizadas
* **Mobile:** React Native (TypeScript) & Expo
* **Backend:** Supabase (PostgreSQL)
* **IoT / Hardware:** ESP32-C3 SuperMini, SIM868/A7680C (GNSS/GSM), C++
